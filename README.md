# VC Intelligence Platform

A lightweight VC sourcing tool built with Next.js App Router.

## Features

- Company discovery with search, sorting, and pagination
- AI-powered website enrichment
- Thesis-based scoring system
- Saved searches (localStorage)
- Custom company lists with export (JSON/CSV)
- Investment thesis configuration
- Fully client-side persistence

## Tech Stack

- Next.js 16 (App Router)
- OpenAI API
- Tailwind CSS
- LocalStorage

## Setup

1. Install dependencies:
   npm install

2. Add environment variable:
   OPENAI_API_KEY=your_key_here

3. Run locally:
   npm run dev
